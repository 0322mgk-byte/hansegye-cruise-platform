# 크루즈 상품별 시각 아이덴티티 가이드

각 크루즈 상품의 대표적인 시각 요소를 정리한 레퍼런스. 프롬프트 생성 시 해당 상품의 색감, 분위기, 배경을 정확하게 반영하기 위해 사용한다.

---

## alaska-emerald-princess (알래스카)

**선박**: 에메랄드 프린세스호 (Emerald Princess)
**항로**: 밴쿠버 → 주노 → 스캐그웨이 → 엔디캇 암 → 케치칸

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 타이드워터 빙하, 설산, 침엽수 피오르드, 깊은 청색 바다, 야생 동물(흰머리독수리, 고래) |
| **색감** | 딥 블루, 아이스 화이트, 에메랄드 그린, 슬레이트 그레이 |
| **분위기** | 장엄(majestic), 고요(serene), 경외감(awe-inspiring) |
| **추천 시간대** | golden hour (산봉우리에 걸리는 금빛), 새벽 안개, 부드러운 오후 빛 |
| **추천 color palette** | `cool blue and white arctic palette with deep emerald undertones` |

### 대표 랜드마크
- Glacier Bay / Endicott Arm (빙하가 바다에 부서지는 장면)
- Inside Passage 피오르드 (양쪽 설산 사이 좁은 수로)
- Mendenhall Glacier (주노 빙하)
- Creek Street, Ketchikan (수상 목조 건물 거리)
- Skagway 골드러시 역사 거리

### 추천 소품/스타일링
- 두꺼운 니트/캐시미어 숄, 따뜻한 코트
- 핫초코, 커피 머그
- 나무 데크 발코니, 금속 레일링

---

## arabia-toscana (아라비아)

**선박**: 코스타 토스카나호 (Costa Toscana)
**항로**: 두바이 → 오만(무스카트) → 카타르(도하) → 아부다비

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 사막 모래언덕, 금빛 모스크, 전통 수크(시장), 초현대식 스카이라인, 페르시안 걸프 바다 |
| **색감** | 앰버, 골드, 따뜻한 샌드, 테라코타, 터콰이즈(바다) |
| **분위기** | 이국적(exotic), 럭셔리(luxurious), 신비로운(mystical) |
| **추천 시간대** | golden hour (사막 위 금빛), magic hour (블루+골드 그라데이션), 밤 스카이라인 |
| **추천 color palette** | `warm amber and gold desert palette with turquoise gulf accents` |

### 대표 랜드마크
- Burj Khalifa / Dubai skyline (두바이 스카이라인)
- Sheikh Zayed Grand Mosque (아부다비 대모스크)
- Dubai Souks (금시장, 향신료 시장)
- Desert Safari (사막 사파리, 모래언덕)
- Souq Waqif, Doha (도하 전통 시장)

### 추천 소품/스타일링
- 가벼운 린넨/실크 소재, 밝은 톤 의상
- 아랍식 차(카락 차이), 대추야자
- 화려한 아라비안 디테일 (아치, 모자이크 패턴)

## north-europe-diadema (북유럽)

**선박**: 코스타 디아데마호 (Costa Diadema)
**항로**: 함부르크 → 코펜하겐 → 게이랑에르 → 베르겐 → 스타방에르

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 노르웨이 피오르드, 폭포, 초록 언덕, 중세 목조 항구 도시, 색색의 나무 건물 |
| **색감** | 쿨 그린, 슬레이트 그레이, 소프트 블루, 우드 브라운, 파스텔 (항구 건물) |
| **분위기** | 평화로운(peaceful), 목가적(pastoral), 청량한(refreshing) |
| **추천 시간대** | 부드러운 오후 빛, 흐린 하늘의 디퓨즈드 라이트, 여름 백야 |
| **추천 color palette** | `cool Scandinavian green and slate palette with soft pastel accents` |

### 대표 랜드마크
- Geirangerfjord (유네스코 피오르드, 칠자매 폭포)
- Bryggen, Bergen (한자동맹 목조 건물 거리)
- Nyhavn, Copenhagen (색색의 타운하우스 운하)
- Pulpit Rock / Preikestolen (절벽 전망대)
- Stavanger 올드타운 (백색 목조 가옥)

### 추천 소품/스타일링
- 니트 스웨터, 트렌치코트, 스카프
- 커피, 와플
- 목조 데크, 빈티지 목재 느낌

## okinawa-taiwan-bellissima (오키나와/대만)

**선박**: MSC 벨리시마호 (MSC Bellissima)
**항로**: 대만 → 미야코지마 → 나하 → 이시가키 → 기륭

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 산호초 바다, 백사장, 열대 식생, 류큐 전통 건축, 붉은 홍등 거리 |
| **색감** | 터코이즈, 에메랄드 그린, 산호 핑크, 선명한 블루 스카이 |
| **분위기** | 밝고 활기찬(bright and vibrant), 열대(tropical), 여유로운(relaxed) |
| **추천 시간대** | 밝은 한낮 (vivid blue sky, crisp shadows), 맑은 아침 |
| **추천 color palette** | `warm saturated tropical palette with turquoise and emerald tones` |

### 대표 랜드마크
- Miyakojima beaches ("오키나와의 몰디브", 에메랄드빛 바다)
- Shuri Castle, Naha (슈리성, 류큐 왕국 유산)
- Ishigaki limestone cave (이시가키 종유석 동굴)
- Jiufen, Taiwan (지우펀 홍등 골목, 천과 천 애니 영감)
- Yehliu Geopark (야류 지질공원, 기암 해안)

### 추천 소품/스타일링
- 가벼운 여름 원피스, 린넨 셔츠, 밀짚 모자
- 아이스 트로피컬 주스, 열대 과일
- 밝은 색감, 꽃무늬 패턴

## southern-med-fortuna (남부 지중해)

**선박**: 코스타 포르투나호 (Costa Fortuna)
**항로**: 이스탄불 → 미코노스 → 헤라클리온 → 로도스 → 산토리니 → 아테네

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 산토리니 블루돔, 하얀 절벽 마을, 에게해 코발트 바다, 올리브 밭, 고대 유적 |
| **색감** | 코발트 블루, 순백, 테라코타, 올리브 그린, 골드 |
| **분위기** | 로맨틱(romantic), 클래식(classic), 따뜻한 지중해(warm Mediterranean) |
| **추천 시간대** | golden hour (하얀 건물에 반사되는 금빛), 석양 (바다 위 오렌지빛) |
| **추천 color palette** | `Mediterranean cobalt blue and white palette with golden terracotta warmth` |

### 대표 랜드마크
- Santorini Oia blue domes (산토리니 이아 블루돔 마을)
- Hagia Sophia, Istanbul (아야소피아)
- Mykonos windmills (미코노스 풍차)
- Acropolis, Athens (아테네 아크로폴리스)
- Rhodes medieval old town (로도스 중세 구시가)

### 추천 소품/스타일링
- 하얀 린넨 의상, 가벼운 면 소재, 선글라스
- 와인 글라스, 올리브, 지중해 과일
- 돌담, 부겐빌레아 꽃, 파란 문/창문

## western-med-seaview (서부 지중해)

**선박**: MSC 씨뷰호 (MSC Seaview)
**항로**: 로마 → 팔레르모 → 이비자 → 바르셀로나 → 마르세유 → 제노아

### 시각 요소
| 항목 | 내용 |
|------|------|
| **대표 풍경** | 유럽 해안 도시, 이탈리아 건축, 스페인 활기, 남프랑스 항구, 지중해 해안선 |
| **색감** | 따뜻한 오렌지, 테라코타, 올리브 그린, 라벤더, 아주르 블루 |
| **분위기** | 활기찬(lively), 문화적(cultural), 세련된(sophisticated) |
| **추천 시간대** | 오후 빛 (따뜻한 사이드 라이팅), golden hour, 매직아워 |
| **추천 color palette** | `warm European terracotta and olive palette with azure coastal tones` |

### 대표 랜드마크
- Sagrada Familia, Barcelona (사그라다 파밀리아)
- Colosseum, Rome (콜로세움)
- Vieux-Port, Marseille (마르세유 구항구)
- Palermo Cathedral (팔레르모 대성당)
- Ibiza Old Town (이비자 구시가, Dalt Vila)

### 추천 소품/스타일링
- 세련된 캐주얼 — 리넨 블레이저, 선드레스, 에스파드리유
- 에스프레소, 와인, 타파스
- 테라코타 타일, 철제 발코니, 카페 테라스
