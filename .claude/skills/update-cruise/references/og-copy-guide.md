# OG Copy Guide — 소비자 욕구 레퍼런스

meta.ts의 `ogTitle`/`ogDescription` 작성 전에 이 파일을 읽고, 소비자 심리를 기반으로 카피를 작성한다.

---

## 1. 타겟 프로필

20~40대 한국인. 해외여행 경험 있으나 크루즈는 처음이거나 1-2회. SNS/카카오톡에서 공유 링크를 본다.

---

## 2. 소비자 욕구 (Desires)

| ID | 욕구 | 왜 효과적인가 | 카피 활용법 |
|----|------|-------------|-----------|
| D1 | 짐 안 싸는 여행 | 자유여행의 최대 스트레스(매일 숙소 이동, 짐 끌기)를 제거. "짐 한 번 풀면 끝" | 자유여행의 고통을 대비시켜 크루즈의 편리함 부각 |
| D2 | 자고 나면 다른 나라 | "눈 뜨면 다른 항구" — 이동 자체가 경험이 되는 마법 같은 감각 | 구체적 장면으로 호기심 유발 ("아침마다 다른 나라에 배달됨") |
| D3 | 가성비 올인클루시브 | 숙박+교통+삼시세끼+공연 포함. 자유여행보다 싸을 수 있다는 반전 | "비싸다"는 편견을 깨는 숫자/비교 활용 |
| D4 | 계획 스트레스 제로 | 기차표, 식당 예약, 숙소 체크인/아웃 전부 불필요 | 여행 계획 세우기 지친 사람에게 소구 |
| D5 | 여러 나라 효율 탐방 | 한 번의 여행으로 4-6개국. 휴가가 짧은 직장인에게 최적 | "N개국을 한 번에"보다 "휴가 아끼면서" 프레임이 효과적 |
| D6 | SNS 인증샷 | 바다 위 수영장, 발코니 석양, 매일 다른 배경 — 비주얼 임팩트 | 구체적 장면 묘사로 "나도 저기 가고 싶다" 유발 |

## 3. 소비자 불안/오해 (Barriers)

| ID | 불안/오해 | 실제 | 후킹 전환법 |
|----|---------|------|-----------|
| B1 | "크루즈는 비싸다" | 1인 40만원대 가능. 숙박+식사 포함이라 자유여행보다 싸을 수 있음 | 가격 반전으로 클릭 유도 ("자유여행보다 싸다고?") |
| B2 | "노인 여행 아닌가" | MZ세대가 크루즈 성장 주도. 2030 전용 상품까지 등장 | 편견 깨기 패턴 ("크루즈 = 황혼여행? 요즘 2030이 다 탄다") |
| B3 | "해상일 지루할 것 같다" | 해상일이 "크루즈의 꽃". 수영장, 스파, 공연, 카지노, 파티 | 해상일의 반전 매력으로 궁금증 유발 |
| B4 | "기항지 시간 너무 짧다" | 6-8시간이지만 핵심만 집중 관광. 이동 피로 없이 다음 날 새 도시 | 오히려 "핵심만 쏙" 프레임으로 전환 |
| B5 | "정장 입어야 하나" | 의무 아님. 갈라 디너도 안 가면 그만 | 격식 부담 해소 |
| B6 | "멀미 걱정" | 대형 선박은 거의 안 흔들림. 대부분 금방 적응 | — |

## 4. 후킹 기법 (Hooking Techniques)

| 기법 | 패턴 | 설명 |
|------|------|------|
| 편견 깨기 | "{오해}? {반전}" | B1~B3의 편견을 정면으로 깨는 방식 |
| 구체적 장면 | "{오감 이미지}" | 추상적 설명 대신 눈에 보이는 한 장면 묘사 |
| 비교 프레임 | "{기존 방식} vs {크루즈}" | 자유여행/패키지의 고통 대비 크루즈의 편리함 |
| 인사이더 | "{행동}하는 법" | 아는 사람만 아는 정보 느낌 |
| 질문 훅 | "{타겟의 고민}?" | 공감으로 시작해서 답을 클릭하게 만듦 |
| FOMO | "{의외의 사실}" | 모르면 손해인 정보 프레임 |

## 5. 상품 특성 → 욕구 매칭

| 상품 특성 | 1순위 욕구 | 접근법 |
|----------|----------|--------|
| 다국가 경유 (지중해, 유럽) | D5 효율 탐방 + D1 짐 편리 | 자유여행 대비 편리함, 다국가 효율 강조 |
| 자연/경관 (알래스카, 북유럽) | D6 인증샷 + D2 매일 새 풍경 | 압도적 비주얼 장면 묘사 |
| 이국적 목적지 (아라비아) | D2 새 나라 배달 + B1 가격 반전 | 접근 어려운 곳을 편하게 가는 프레임 |
| 단기 일정 (오키나와 5박) | D4 계획 제로 + D3 가성비 | 짧은 휴가에 최적화, 가격 소구 |
| 럭셔리 선박 | D6 인증샷 + B2 편견 깨기 | "이 가격에 이런 배를?" 반전 |

## 6. 금지 패턴 (Anti-Patterns)

- **도시 나열**: "A · B · C를 한 번에!" → `shortDescription`이 이미 하는 역할
- **상품명 복사**: `name`이나 `cardTitle`을 그대로 쓰는 것
- **뻔한 수식어**: "최고의", "꿈의", "환상적인", "특별한 경험"
- **애매한 약속**: "잊지 못할 추억", "일생일대의 여행"
- **관광지 정보 요약**: Agent P 조사 결과를 압축한 것 (→ 그건 상품 페이지 안에서 할 일)
